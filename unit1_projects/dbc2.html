Cristobal's Path to Coding Intermediacy.
So you want to learn to code.

If you pick the blue pill, the story ends, and you can live blissfully in coding ignorance. If you pick the red pill, I will show you how deep the rabbit hole goes.

If you are the type to want to learn it for yourself for free, there are plenty of great options for you to do just that. Codecademy.com is completely free, with message boards to help you when you get stuck.

If you're looking for something a bit more meaty and don't mind paying a little for a more premium service, check out Teamtreehouse.com, which has plenty of excellent videos, excercises, quizzes, and can help you from the ground up.

Finally, if you're like me, and really need that fire lit under you to learn effectively, apply to Dev Bootcamp! A 9-week intensive in locations across 3 great, American cities, DBC can get you where you need to be!
