Wireframe sites

Facebook


The web page is solving the problem of me not knowing what my friends are thinking, eating, and doing. It allows for a steady stream of information about my friends to be presented before me in an easily consumable fashion. If I want to know something specific, there are ways to look up friends, events, or whatever else I might need.

Dark Roasted Blend


Dark Roasted Blend solves the problem of boredom. It presents many interesting images for me to click on and I can go from one to the next in succession based on my interest in the moment.

Codecademy


Codecademy provides a service to learn programming languages. The visual approach is straightforward and allows the user to clearly see the various languages offered as well as his/her progress completing the track for that language.
